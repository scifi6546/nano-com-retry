# GPU API

Memory 0x100000 bytes

1 modes:
256x256 16 colors
each pixel is 4 bits long

input signal
phase 1.
```
*------------------------------*
|x: (8 bits)  | y: (8 bytes)  | 
*------------------------------*
```
phase 2.
```
*----------------------------------------------*
|color:(4 bytes)  | draw duration (12 bytes)   |
*----------------------------------------------*
```
